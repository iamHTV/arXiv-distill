# Skill: Autonomous Business Model Strategy
Source: 2506.17339
Type: framework

## When to use this skill
- When designing AI-first business strategy
- When evaluating transition from AI-augmented to AI-autonomous
- When preparing for synthetic competition

## When NOT to use
- AI capability not yet sufficient to execute core functions
- Organization not ready for autonomous operations
- Regulatory environment restricts AI autonomy

## Required inputs
- Current business model documentation
- AI capability assessment
- Competitive landscape analysis
- Data infrastructure evaluation

## Steps

### Step 1 — Assess Current State
1. Classify: human-driven, AI-augmented, or AI-autonomous
2. Identify core value creation, delivery, capture mechanisms
3. Evaluate: which mechanisms can AI execute today?
4. Output: current state assessment

### Step 2 — Design ABM Architecture
1. Identify autonomous execution opportunities
2. Design continuous adaptation mechanisms
3. Plan gradual offloading of human decisions
4. Output: ABM architecture blueprint

### Step 3 — Build AI-Native Competitive Advantages
1. Identify machine-level speed opportunities
2. Design synthetic competition responses
3. Build AI-native moats
4. Output: competitive strategy

### Step 4 — Implement Gradually
1. Start: low-stakes autonomous decisions
2. Expand: medium-stakes decisions
3. Scale: high-stakes strategic decisions
4. Monitor: performance, risks, governance

## Expected output
- ABM architecture blueprint
- Competitive strategy for synthetic competition
- Gradual implementation roadmap
- Risk management framework

## Example application
**Scenario**: Traditional retailer wants to transition to AI-autonomous.

**Application**:
1. Current: human-driven inventory, pricing, customer service
2. ABM design: AI manages inventory autonomously, adjusts prices real-time, handles customer relationships
3. Competitive advantage: AI negotiates with suppliers 24/7, optimizes pricing 1000x/day
4. Gradual: start with inventory → expand to pricing → expand to supplier negotiations
5. Result: synthetic competition ready, machine-level speed

## Gotchas
1. **AI IS strategy, not tool**: Don't just augment. Rethink entire business model
2. **Synthetic competition is coming**: Machine-level speed. Prepare now
3. **Gradual offloading**: Start low-stakes, expand. Don't go autonomous overnight
4. **Governance needed**: Autonomous decisions need oversight frameworks
5. **Human role changes**: From decision-maker to oversight/governance
6. **Timeline uncertain**: ABMs are emerging. Start experimenting now
