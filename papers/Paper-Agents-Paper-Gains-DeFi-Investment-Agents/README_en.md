# [2605.29174v1] Paper Agents, Paper Gains: An Empirical Analysis of DeFi Investment Agents

## Problem
DeFi investment agents — systems using AI for autonomous on-chain trading — have attained over $3 billion in combined token valuations since late 2024. However, no empirical analysis has evaluated: (1) how many projects are truly autonomous vs basic API integrations; (2) who actually benefits from agents — developers or token holders; (3) whether token valuations are connected to fundamentals.

## Contribution
1. **Survey of 1,900 AI-tagged crypto projects**: Filtered to 10 representative projects, deep-dive architectural analysis of ElizaOS and Virtuals Protocol.
2. **On-chain performance analysis**: 11 Solana-based agent treasuries, 925,323 token holders. Found: agent treasuries retain $30M+ paper gains while token holders collectively lost $191.7M.
3. **Maturity framework**: 3 dimensions — autonomous execution, risk-adjusted profitability, stakeholder alignment.

## Method (core summary)
Survey 1,900 AI-tagged crypto projects → filter investment-focused agents → curate 10 representative projects. Deep-dive: ElizaOS and Virtuals Protocol architectures. Quantitative on-chain analysis: 11 Solana agent treasuries, 925,323 token holders, track trading activity, gains/losses, token performance. Propose maturity framework: autonomous execution (truly autonomous?), risk-adjusted profitability (returns justify risk?), stakeholder alignment (who benefits?).

## Key Findings
- Top 1% of wallets capture 81.4% of ALL gains ($1.81B) — extreme wealth concentration
- Agent treasuries retain $30M+ paper gains, token holders collectively lost $191.7M
- Market-cap-to-AUM ratios: >10,000x for agent tokens vs <1x for established DeFi protocols
- User gains peaked at $2.4B → declined to net losses, median returns NEGATIVE on every platform
- Tokens declined 93% average from all-time highs
- Many projects provide no clear evidence of autonomous trade execution — basic API integrations
- Developer interviews: many visible deployments remain basic API integrations

## Actionable Insights
1. **Evaluate AI agent projects using maturity framework**: When evaluating AI agent investment, check 3 dimensions: (a) autonomous execution proof — or just API wrapper? (b) risk-adjusted profitability — returns justify risk? (c) stakeholder alignment — who benefits? Example: before investing in AI trading agent, require proof of autonomous execution (on-chain evidence), risk-adjusted returns (Sharpe ratio), and transparent stakeholder alignment.

2. **Token valuation ≠ fundamentals**: Market-cap-to-AUM >10,000x = pure speculation. When evaluating DeFi agent tokens, compare market cap with actual AUM and revenue. Example: if agent token has $100M market cap but only $10K AUM → valuation disconnected, avoid.

3. **Top 1% captures 81.4% of gains — early market dynamics**: Permissionless markets enable rapid experimentation but also allow wealth concentration. When participating, be aware that early movers/insiders capture majority of gains. Example: if entering AI agent crypto market, expect extreme inequality — don't assume fair distribution.

## Assumptions & Limitations
**Conditions for analysis to work:**
- On-chain data must be publicly attributable
- Solana-based projects — ecosystem-specific
- Survey methodology: AI-tagged projects may miss non-tagged projects

**Limitations the paper acknowledges:**
- Curated sample — not representative of all projects
- Many projects early stage — insufficient data for long-term analysis
- On-chain analysis limited to publicly attributable activity

**Limitations the paper does NOT mention:**
- No comparison with non-AI DeFi agents — is this AI-specific problem?
- Solana ecosystem only — Ethereum, Base, others may differ
- Survey methodology opaque — filter criteria unclear
- No regulatory implications addressed
- No developer incentive analysis — why launch agents?
- 93% token decline may be due to market-wide crypto downturn, not agent-specific
